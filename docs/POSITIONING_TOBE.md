## 1. 한 줄

**하루에 한 장, 모르는 사람에게 가벼운 질문을 띄우고 — 답이 오고 마음이 오면 그때 얼굴이 열린다.**

스와이프 데이팅 앱이 아니다. **희소 미션 + 말 먼저 + 짧은 설렘** 습관 앱이다.

---

## 2. 시장에서의 자리

| 축 | As-Is (MVP) | To-Be |
|----|-------------|--------|
| 카테고리 | Social Networking (스토어) | 유지 — Dating 카테고리 진입 금지 |
| 경쟁 프레임 | (암묵) 매칭 앱과 비교되기 쉬움 | **스와이프 피로 대안** / “하루 한 번 설렘” |
| 약속 | 연애 성사 | 소소한 도파민 · unlock 경험 |
| KPI | (미계측) | 주간 unlock · D1 · (보조) 답장률 |
| 성역할 | 여 발송 / 남 수행 가설 | 베타로 검증 후 유지·완화·스왑 결정 |

**하지 않을 것:** 무한 피드, 스펙 필터 매칭, 영구 메신저, “이상형 찾기” 카피.

---

## 3. 차별화 기둥 (북극성에 고정)

1. **말 먼저 · 얼굴 나중** — 질문·답장이 관심 신호. 프로필은 보상.  
2. **하루 한 장 희소성** — 랜덤 · 타이머 · 일일 캡.  
3. **Sea 세계관** — 병·쪽지·표류. 카드 스와이프 UI 금지.  
4. **짧은 채팅 창** — 7일 · 약속/연락처 유도. 메신저 대체 아님.  
5. **여 발신 안전 구조** — 불쾌 미션 유입을 역할로 줄임 (가설, 검증 대상).

AI 소개·칩·연예인 닮은꼴 등은 **보조재**. 루프 감정을 대체하지 않는다.

## 4. 타깃 · 톤

| | 내용 |
|--|------|
| Primary | 스와이프에 지친 20대 중후반~30대, 가벼운 설렘을 원함 |
| 여성 | “부담 없는 한 질문”을 띄우는 쪽 — 안전·톤이 최우선 |
| 남성 | “발견의 재미” — 빈 바다에서 이탈하지 않게 |
| 톤 | 따뜻·담백·놀이. 무거운 연애 코칭·스펙 비교 금지 |

---

## 5. 성공 / 실패 정의 (To-Be)

**성공:** 주간 unlock이 안정적으로 나오고, 여 D1이 버티며, “데이팅 앱 같다”보다 “하루 한 장 설렘”으로 회자됨.

**실패:** 풀 부족으로 빈 바다 · 여성 이탈 · 스와이프형 기능 추가 압박에 굴복 · KPI를 매칭 수로만 최적화.

---

## 6. 포지션 문장 (대외)

- KR: *가벼운 질문이 바다를 떠요. 마음이 오면, 그때 열려요.*  
- EN: *A light question drifts out. When it clicks, profiles open.*

스토어·온보딩·푸시 카피는 이 문장의 변주만 쓴다.
