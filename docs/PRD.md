## 0. 제품 감각 (북극성)

기능은 최대한 단순하게. 대신 감정은 매일 조금:

- **타이머** — 수락 후 미응답 **12h** 만료 (긴장감)
- **랜덤 뽑기** — 누가 받을지 / 누구에게 올지 모름
- **일일 한 장** — 하루 1회 무료 발송 (희소성)
- **양방향 OK** — unlock 전 설렘 유지
- **짧은 대화** — 매칭 후 무제한 메신저가 아님 (조건 설계 중)

북극성 KPI: **주간 양방향 OK(unlock) 수** + D1 재방문.

## 3. 역할 · 성별 규칙 (가설 · 검토 중)

| 역할 | 누가 | 하는 일 |
|------|------|---------|
| **발송** | **여성만** | 미션/쪽지 작성·전송, 답장 평가(OK/패스) |

## 4. 핵심 루프

> **SSOT:** 코드(Sea `/home` + `20260717*` 마이그레이션). 구 “양방향 OK” 모델은 폐기.

```text
(여) 질문 띄우기 → 랜덤 남 배달 → 표류
 → (남) 발견 · 나이대/지역 선확인
      ├─ 패스(열기 전) → 노패널티 · 14일 쿨다운 · **자동 재배달**(≤5) · 여 알림
      └─ 열어보기 → 수락 → 12h 내 답장
           ├─ 포기/만료 → 남 **24h 수신 금지** · 여 재발송 제안
           └─ 답장 (=관심 신호)
 → (여) 마음에 들어요 → 프로필 unlock
 → 티켓 1장으로 매칭 → 인앱 채팅(**메시지 무제한 · 7일**)
 → 최초 띄움 후 48h 미수락 → 회수 · 여 재작성 유도
```

| 규칙 | 내용 |
|------|------|
| 열기 전 (남) | 나이대·지역만. **패스 = 노패널티 + 자동 재배달** |
| 열기 후 (남) | 12h 타이머. 미답장(만료/포기) = **하루 동안 플로티 수신 불가** |
| 남 슬롯 | **동시 1** — 수행 중·유효 채팅 중이면 수신/추가 매칭 불가 |
| 여 슬롯 | 띄운 플로티 수만큼 **병렬 채팅 OK** · 활성 채팅 중 **신규 플로티 잠금** |
| 답장 | 관심 신호. 텍스트 + 사진 첨부(선택). 여가 「사진 부탁」을 켜도 **강제 아님** |
| Unlock | **여 OK**만으로 프로필 오픈 (남 답장 전제) |
| 매칭 | 티켓 **1장** (먼저 누른 쪽). 단가 **₩4,900** |
| 채팅 | **통수 무제한 · 기한 7일**. 그 안에 약속·연락처 교환. 추후 티켓 연장 |
| 이상형 필터 | **Out** (쓰지 않음) |
| 도착 알림 | in-app (push 후속) |

→ [`decisions/0010-concurrency-and-redeploy.md`](./decisions/0010-concurrency-and-redeploy.md)

## 6. 랜덤 풀 · 매칭

**활성 우선 랜덤 + 단계적 확대 + 슬롯 제외.**

1. **1순위 풀:** 최근 **48시간** 접속한 이성(남)  
2. 부족 시 → **7일** → **30일** / 전역  
3. 하드 제외: 차단, 페어 쿨다운, 일일 수신 캡, 미온보딩, **`receive_blocked_until`**  
4. **슬롯 제외:** 플로티 수행/대기 중(`delivered`·`replied`) · 유효 채팅 중 · 동일 미션 기수신자

→ [`decisions/0002-matching-weak-fit.md`](./decisions/0002-matching-weak-fit.md) · [`0010-concurrency-and-redeploy.md`](./decisions/0010-concurrency-and-redeploy.md)

## 7. Unlock 후 프로필 공유

**방침:** A 자체 미니 프로필. Palette는 후순위.  
**프로필 사진:** **3장 필수** (온보딩·수정·발송/수신 풀 공통). 플레이스홀더·미업로드 계정은 풀에서 제외.

**고도화 (Accepted):** unlock 카드 = **나 ~70% + 끌리는 사람 ~30%** (AI 요약 1문장).  
온보딩: S1–S4 + I1·I2 필수 · 직업/흡연 · 키 선택. 발송용 이상형 필터는 Out.

→ 상세: [`PROFILE_INTERVIEW_V2.md`](./PROFILE_INTERVIEW_V2.md) · [`decisions/0007-post-unlock-profile-open.md`](./decisions/0007-post-unlock-profile-open.md)

## 9. MVP In / Out

**In**

- 여 발송 / 남 수행 (가설)
- 하루 1회 무료 발송, 추가 과금 훅
- 48h 활성 우선 랜덤 + 확대
- 텍스트 미션 · 사진 부탁(soft) · 여 OK unlock · 타이머
- 자체 미니 프로필 unlock
- 인앱 채팅 무제한 · 7일
- 차단·신고·안전 센터

**Out (초기)**

- 공개 피드·팔로우·AI 유사도 (레거시)
- 장소 집합 미션
- 이상형 필터
- Palette 강제 연동
- 영구·무기한 채팅앱 · 광고 피드
