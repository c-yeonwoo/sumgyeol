## 1. 필드

### 팩트 (온보딩 1스텝 · 전부 필수)

| 필드 | 필수 | 노출 |
|------|------|------|
| 하는 일 칩 | ✅ (+전문직) | 헤더 메타 |
| 흡연 | ✅ (`안 함`·`가끔`·`함` · 비공개 Out) | 짧은 라벨 |
| 음주 | ✅ | 짧은 라벨 |
| 타투 | ✅ | 짧은 라벨 |
| 종교 | **보류** (민감·카테고리 리스크) | — |
| 키 | ✅ · 인라인 `cm` | 헤더 `170cm` |

Unlock 카드: Q&A 섹션 Out · AI 소개는 `##` 챕터 스토리(+사진 인터리브).

흡연 옵션: `안 함` · `가끔` · `함` · `비공개`

하는 일: `직장인` · `학생` · `프리랜서` · `창업` · `기타`

### 나 (S1–S4)

| ID | 질문 | 입력 |
|----|------|------|
| S1 | 요즘 빠져 있는 건 뭐예요? | 자유 텍스트 (단답 OK) |
| S2 | 쉬는 날은 보통 어떻게 보내요? | 자유 텍스트 |
| S3 | 주변에서 자주 듣는 말은요? | 자유 텍스트 |
| S4 | 연애에서 더 가까운 쪽은? | 칩 1 필수 |

S4: `천천히 깊게` · `편하고 즐겁게` · `서로 성장` · `자주 연락하며` · `아직 모르겠어요`

AI 생성 시 **팩트 + S1–S4 + I1–I2 전체**를 넣고, 단답은 창작 없이 연결해 챕터로 푼다.

### 이상형 입력 (I1–I2 · 필수)

| ID | 질문 | 입력 |
|----|------|------|
| I1 | 끌리는 사람의 분위기는? | 칩 ≤2 |
| I2 | 같이 있으면 좋은 리듬은? | 칩 1 |

I1: `잔잔한` · `유머 있는` · `진지한` · `따뜻한` · `솔직한` · `여유로운` · `호기심 많은`  
I2: `천천히 알아가기` · `자주 연락` · `주말 위주` · `즉흥 만남` · `아직 모르겠어요`

칩 소스는 `app_config` JSON (`interview_chips_v2`) — 코드 폴백 + 원격 오버라이드(어드민 UI는 후속).

### AI 산출

- `ai_intro` — 나 2–3문장  
- `ai_ideal_line` — 끌리는 사람 1문장 (카드에 이것만)  
- `ai_tags` — 나 중심 태그

## 2. 온보딩 순서

```text
닉 → 생년/성별 → 지역 → 사진3
 → facts (직업·흡연 필수, 키 선택)
 → S1 → S2 → S3 → S4
 → bridge 「어떤 사람에게 마음이 기울어요?」
 → I1 → I2
 → AI 생성 → 리뷰(intro + ideal_line + tags) → 시작
```

---

## 3. unlock 카드

```text
히어로: 이름·나이대 · 지역 · 키? · 직업 · 흡연(비공개면 숨김)
이런 사람이에요 ← ai_intro
관심사 태그
이런 사람에게 끌려요 ← ai_ideal_line only
몇 가지 질문 ← S1–S3
```
