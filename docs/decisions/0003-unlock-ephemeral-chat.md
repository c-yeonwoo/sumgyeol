# ADR 0003 — Unlock 후 인앱 채팅 (무제한 아님)

- Status: Accepted direction / caps Open
- Date: 2026-07-14

## Context

외부 연락 즉시 교환은 안전·설렘 버퍼가 약하다.  
영구 메신저는 제품이 채팅앱이 된다.

## Decision

- 인앱 텍스트 채팅 **필수**
- 매칭 후 **무제한 채팅 금지**
- 캡 방식은 미정 → 후보: 메시지 수 / 시간 창 / 턴제 / 동의 시 외부 핸드오프

## Interim (구현용 임시값)

확정 전까지: **메시지 20통 또는 7일 중 먼저 도달 시 soft close** + 양측 동의 시 연락처 교환 옵션.

## Open

최종 캡 숫자, 티켓으로 연장 가능 여부, 턴제 채택 여부.
