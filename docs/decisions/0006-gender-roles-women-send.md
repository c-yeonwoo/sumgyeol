# ADR 0006 — 성별 역할: 여성 발송 / 남성 수행

- Status: Proposed (검토 중)
- Date: 2026-07-14

## Context

커스텀·행동 미션을 누구나 보내면, 일부 남성 발송자가 불쾌·성적·이상한 미션을 뿌릴 위험이 크다.  
모더레이션만으로 막기엔 비용이 크고, 초기 앱의 톤이 쉽게 오염된다.

## Decision (가설)

- **미션 발송: 여성만**
- **미션 수행(수신·답장): 남성만**

스토어·법적·성비 수급을 보며 재검토한다. 확정 전에도 스키마/가드는 이 방향으로 구현해 실험한다.

## Consequences

- 남 유저의 “보낼 재미”가 없어짐 → 수행·OK·unlock·채팅에서 도파민을 따로 설계해야 함
- 여 유저 부족 시 송신이 말라 전체 루프가 멈춤 → 초대·대기열 UX 필요
- 앱이 “이성 설렘”에 더 가까워 보여 스토어 카피와 긴장 관계
