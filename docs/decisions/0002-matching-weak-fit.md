# ADR 0002 — 매칭: 활성 우선 랜덤 + 단계 확대

- Status: Accepted (v2) — 슬롯 제외는 [`0010`](./0010-concurrency-and-redeploy.md)
- Date: 2026-07-14
- Supersedes: v1 약한 fit 서술

## Context

완전 랜덤은 죽은 계정에 떨어지고, 딥 fit은 데이팅앱화한다.  
도파민은 “랜덤 뽑기”에서 오므로 규칙은 설명 가능해야 한다.

## Decision

1. 하드: 역할(여→남), 18+, 차단, 쿨다운 14일, 수신 캡  
2. 풀 확대:
   - 1순위: 최근 **48h** 접속
   - 부족 시: 최근 **7일**
   - 그래도 부족: **30일** → 전역 (파라미터)
3. 이상형 조건: **1개 무료**, 2개+ 티켓  
4. ML 취향 fit 없음

## Consequences

- 활성 유저가 미션을 더 자주 받음 (의도)
- 심야/비활성 시간대는 확대 풀로 폴백 → UI에 “멀리/오래 접속” 힌트는 unlock 후에만
