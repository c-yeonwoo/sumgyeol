# ADR 0007 — Unlock 후 프로필 공유 (작성 방식 미정)

- Status: Open
- Date: 2026-07-14

## Context

양방향 OK 후 “프로필 공유”가 설렘의 보상이다.  
자체 작성 vs Palette 연동 vs 하이브리드가 후보.

## Options

| | A 자체 | B Palette 연동 | C 하이브리드 |
|--|--------|----------------|--------------|
| 속도 | 빠름 | 연동 비용 | 중간 |
| 신뢰 | 약함 | 강함 | 선택적 |
| 결합 | 없음 | 계정·브랜드·정책 | 선택 결합 |

## Interim

MVP는 **A 자체 미니 프로필** (닉네임, 한 줄, 연령대, 지역, 선택 사진 1).  
Palette 연동은 런칭 이후 soft 검토. 초기 “완전 분리” 원칙과 충돌하므로 기본값으로 넣지 않는다.

## Open

필드 목록, 작성 타이밍(가입 시 vs unlock 직전 강제), 사진 필수 여부.
