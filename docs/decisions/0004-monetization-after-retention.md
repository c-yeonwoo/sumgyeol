# ADR 0004 — 수익화: 일일 1회 + 티켓

- Status: Accepted (v2)
- Date: 2026-07-14

## Context

희소성이 도파민이다. 무제한 발송은 스팸·톤 붕괴.  
초기 “완전 무료”는 루프 검증용으로만 유효했고, 제품 규칙은 유료 훅을 전제로 한다.

## Decision

| 과금 포인트 | 규칙 |
|-------------|------|
| 미션 발송 | **하루 1회 무료**, 추가분은 티켓 |
| 이상형 필터 | **1개 무료**, 2개부터 티켓 |
| 수신·답장·OK·unlock | **무료** (오염 금지) |
| 채팅 연장 | 캡 초과 시 티켓 후보 (채팅 ADR과 연동) |

광고 없음. 티켓 가격·패키지는 런칭 직전 확정.

## Consequences

- 여 유저가 하루 한 번 “뽑기”를 기다림 → 리텐션 훅
- 페이월은 발송/필터에만 걸려 핵심 설렘(OK)은 막지 않음
